# npm-start

## How to copy
```bash
$ git clone [this_repository] [local_project_name]
```

## How to install
```bash
$ cd [local_project_name]
$ npm i
```
## How to run development
```bash
$ npm start
```

## What includes
1. [Imagemin-CLI](https://github.com/imagemin/imagemin-cli)
2. [Pug-CLI]()
3. [Copyfiles](https://github.com/calvinmetcalf/copyfiles)
